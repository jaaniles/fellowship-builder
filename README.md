# Fellowship Builder

A roguelike deck-builder where you lead a fellowship through 10 dangerous segments.

## Quick Start

```bash
# Install dependencies
npm install

# Play the game
npm start
```

## How to Play

1. **Choose a Leader** - Each has unique stats and tactics
2. **Draft Phase** - Pick members, gear, or tactics to strengthen your fellowship
3. **Face Segments** - Your stats are tested against challenges (combat, survival, social, chaos)
4. **Survive** - Complete all 10 segments without losing all HP
